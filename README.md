# Cartão de Crédito Dark

Representação visual de um cartão de crédito com layout escuro, logo, chip, número, titular e validade.

Este projeto é uma demonstração estática feita com **HTML e CSS**, pensada para funcionar diretamente no navegador e poder ser publicada com **GitHub Pages**.

## Funcionalidades

- layout de cartão com header e detalhes
- chip e logo posicionados no topo
- tipografia clara sobre fundo escuro
- responsividade básica

## Estrutura do projeto

- `img/bg.png`
- `img/chip.png`
- `img/icone.png`
- `img/logo.png`
- `index.html`
- `style.css`

## Visual

O projeto usa um cartão escuro com detalhes em branco e imagem de fundo.

## Como executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente no navegador.
3. Se preferir usar um servidor local, execute na pasta do projeto:

```bash
python -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## GitHub Pages

Como este projeto é estático, ele pode ser publicado no GitHub Pages usando a branch principal.

Configuração recomendada:

- Branch: `main`
- Pasta: `/ (root)`

## Tecnologias

- HTML5
- CSS3
